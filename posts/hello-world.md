# 블로그를 시작합니다

안녕하세요. 첫 포스트입니다.

이 블로그는 순수 HTML, CSS, JavaScript로만 만들어졌습니다.
별도의 빌드 과정 없이, 마크다운 파일을 추가하면 글이 올라가는 구조입니다.

## 블로그 구조

- `index.html` — 포스트 목록 페이지
- `post.html` — 포스트 본문 뷰어
- `posts.json` — 포스트 메타데이터 목록
- `posts/*.md` — 실제 포스트 내용

## 새 포스트 추가하는 법

1. `posts/` 디렉토리 안에 `my-new-post.md` 같은 파일을 만들고 마크다운으로 글을 씁니다.
2. `posts.json`의 `posts` 배열 맨 앞에 메타데이터 한 덩어리를 추가합니다.

   ```json
   {
     "slug": "my-new-post",
     "title": "새 포스트 제목",
     "date": "2026-04-22",
     "summary": "한 줄 요약",
     "tags": ["태그1", "태그2"]
   }
   ```

3. 커밋하고 푸시하면 끝.

`slug`는 마크다운 파일명(확장자 제외)과 똑같이 맞춰주세요.

## 지원하는 문법

일반적인 **마크다운** 문법을 모두 지원합니다. *기울임*, `인라인 코드`, 그리고:

```javascript
// 코드 블록도 문법 강조가 됩니다
function hello(name) {
  return `안녕, ${name}!`;
}
```

> 인용문도 깔끔하게 표시됩니다.
